Projekt Ydon – Mobile Stundenbuchung

Über das Projekt

Ydon ist ein Schulprojekt der HTL [Name deiner Schule], Klasse 4BHIF, in Kooperation mit der ITPRO Consulting & Software GmbH in Linz. Ziel ist die Entwicklung einer mobilen App, die als zusätzliches Frontend für das bestehende Stundenbuchungssystem von ITPRO dient.

Mitarbeiter*innen sollen damit ihre Arbeitszeiten ortsunabhängig über Smartphone oder Tablet buchen können. Die App wird über eine REST-API an Microsoft Dynamics 365 angebunden und erweitert damit die bestehende Unternehmenslösung um eine flexible mobile Komponente.

Projektziele

  • Entwicklung einer funktionierenden mobilen App zur Stundenbuchung
  • Anbindung an Dynamics 365 über REST-API
  • Vergleich von Low-Code-Plattformen mit einer klassischen Angular-Umsetzung
  • Bewertung von Benutzerfreundlichkeit, Performance und Wartbarkeit
  • Erstellung einer technischen und vergleichenden Dokumentation

Technologien

  • Microsoft Power Apps (Low-Code)
  • Angular Framework
  • REST-API (Microsoft Dynamics 365)
  • Visual Studio Code, GitHub Pages

Ziel und Nutzen

Mit der mobilen Erweiterung des bestehenden Systems wird die Stundenbuchung für Mitarbeiter*innen der ITPRO deutlich einfacher und flexibler. Das Projekt bietet die Möglichkeit, moderne Entwicklungsmethoden praxisnah zu vergleichen und zu bewerten, welche Technologie sich besser für Unternehmenslösungen eignet.
